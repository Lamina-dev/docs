Gauss-Legendre 求积：将 \([a,b]\) 仿射映射到 \([-1,1]\) 后使用正交节点。

对 2*order-1 次以下的多项式精确积分。节点与权重在编译时预计算（order <= 20）。

参数
[in]func被积函数回调。
[in]user_data传递给 func 的用户上下文。
[in]a积分下限。
[in]b积分上限。
[in]order正交多项式阶数（节点数），范围 [1, 20]。
[out]out_result输出积分近似值。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 order 超出范围或指针为 NULL。
副作用
  • 回调 func 被调用恰好 order 次。
  • 不分配堆内存。

自适应 Simpson 积分，通过递归二分自动控制精度。

递归地将区间二分，直到局部 Simpson 估计满足容差条件或达到最大递归深度。 适用于被积函数局部变化剧烈的情况。

参数
[in]func被积函数回调。
[in]user_data传递给 func 的用户上下文。
[in]a积分下限。
[in]b积分上限。
[in]abs_tol绝对容差（>= 0）。
[in]rel_tol相对容差（>= 0）。
[in]max_depth最大递归深度，超出返回 LMMC_STATUS_WARNING_MAX_DEPTH 。
[out]out_result包含积分值、误差估计与求值次数。
返回
LMMC_STATUS_OK 在容差内收敛； LMMC_STATUS_WARNING_MAX_DEPTH 达到最大深度但结果仍可用； LMMC_STATUS_INVALID_ARGUMENT 若指针为 NULL 或容差为负。
副作用
  • 回调 func 被调用次数取决于递归深度，最坏情况约 \(O(2^{max\_depth})\) 次。
  • 使用递归栈，深度受 max_depth 限制。
  • 不分配堆内存（仅使用栈空间）。

Romberg 积分（Richardson 外推加速梯形法则）。

通过逐次加密梯形法则并进行 Richardson 外推，快速提升收敛阶。 对光滑被积函数效率极高。

参数
[in]f被积函数回调。
[in]ud用户数据指针，传递给 f 。
[in]a积分下限。
[in]b积分上限。
[in]abs_tol绝对容差，范围 [1e-15, 1e-1]。
[in]max_iter最大迭代次数（Romberg 表行数），范围 [1, 1000000]。
[out]out积分结果（值、误差估计、求值次数）。
返回
LMMC_STATUS_OK 在容差内收敛； ::LMMC_STATUS_MAX_ITERATIONS 达到最大迭代次数但结果仍可用； LMMC_STATUS_INVALID_ARGUMENT 若参数超出有效范围或指针为 NULL。
副作用
  • 回调 f 被调用次数记录在 out->num_evals 中，约为 \(2^{max\_iter}\) 量级。
  • 内部分配 Romberg 表所需的临时数组，函数返回前释放。

Tanh-Sinh（双指数）积分，适用于端点奇异性。

利用 \(\tanh(\frac{\pi}{2}\sinh t)\) 变换将端点奇异性映射为指数衰减， 对端点处有代数或对数奇异性的被积函数特别有效。

参数
[in]f被积函数回调。
[in]ud用户数据指针，传递给 f 。
[in]a积分下限。
[in]b积分上限。
[in]abs_tol绝对容差，范围 [1e-15, 1e-1]。
[in]max_nodes最大节点数，范围 [1, 1000000]。
[out]out积分结果（值、误差估计、求值次数）。
返回
LMMC_STATUS_OK 在容差内收敛； ::LMMC_STATUS_MAX_ITERATIONS 达到最大节点数但结果仍可用； LMMC_STATUS_INVALID_ARGUMENT 若参数超出有效范围或指针为 NULL。
副作用
  • 回调 f 被调用次数记录在 out->num_evals 中，不超过 max_nodes 次。
  • 内部分配节点/权重临时数组，函数返回前释放。

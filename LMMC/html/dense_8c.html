◆ lmmc_mat_create()

lmmc_status_t lmmc_mat_create ( size_t  rows,
size_t  cols,
lmmc_mat_t *  out_mat 
)

分配并初始化一个 rows × cols 的稠密矩阵，所有元素置零。

内部通过 lmmc_alloc 分配 rows*cols 个 lmmc_real_t 的连续缓冲区， stride 设为 cols（紧凑存储），owns_data 置 1。

参数
[in]rows行数，必须 > 0。
[in]cols列数，必须 > 0。
[out]out_mat输出矩阵结构体（调用方提供栈/堆上的结构体地址）。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — rows 或 cols 为 0，或 out_mat 为 NULL，或 rows*cols 溢出。
  • LMMC_STATUS_ALLOCATION_FAILED — 内存分配失败。
副作用
  • 分配堆内存（通过 lmmc_alloc），调用方必须配对调用 lmmc_mat_destroy 释放。
  • out_mat 的所有字段被覆写。
线程安全
本函数本身无共享状态，多线程可并发调用（前提是 lmmc_alloc 线程安全）。

◆ lmmc_mat_wrap()

lmmc_status_t lmmc_mat_wrap ( size_t  rows,
size_t  cols,
size_t  stride,
lmmc_real_t *  data,
lmmc_mat_t *  out_mat 
)

用外部缓冲区构造矩阵视图（不拥有内存，不分配）。

将已有的连续内存区域包装为 lmmc_mat_t，owns_data 置 0， 因此 lmmc_mat_destroy 不会释放 data 指针。 调用方需保证 data 的生命周期覆盖该视图的使用期。

参数
[in]rows行数，必须 > 0。
[in]cols列数，必须 > 0。
[in]stride行步距（相邻行首元素间距），要求 >= cols 。
[in]data外部缓冲区指针，至少容纳 rows*stride 个 lmmc_real_t。不可为 NULL。
[out]out_mat输出矩阵视图。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 参数非法。
副作用
  • 无内存分配。out_mat 的所有字段被覆写。
  • 对视图的写操作会直接修改 data 指向的外部缓冲区。

◆ lmmc_mat_destroy()

void lmmc_mat_destroy ( lmmc_mat_t *  mat)

销毁矩阵，释放其拥有的底层缓冲区。

若 mat->owns_data 非零，调用 lmmc_free 释放 mat->data； 否则仅将结构体字段清零。对 NULL 指针或已销毁（data==NULL）的矩阵安全。

参数
[in,out]mat待销毁的矩阵，可为 NULL。
副作用
  • 若 owns_data：释放堆内存，mat->data 置 NULL，rows/cols/stride 清零。
  • 若 !owns_data：仅清零结构体字段，不释放外部缓冲区。
  • 销毁后该矩阵不可再用于任何运算（除非重新 create/wrap）。

◆ lmmc_mat_gemm()

lmmc_status_t lmmc_mat_gemm ( lmmc_real_t  alpha,
const lmmc_mat_t *  A,
int  transA,
const lmmc_mat_t *  B,
int  transB,
lmmc_real_t  beta,
lmmc_mat_t *  C 
)

通用矩阵-矩阵乘法（GEMM）： \(C \leftarrow \alpha \cdot \mathrm{op}(A) \cdot \mathrm{op}(B) + \beta \cdot C\)。

op(X) = X（transX==0）或 X^T（transX!=0）。 设 op(A) 为 M×K，op(B) 为 K×N，则 C 必须为 M×N。 当 LMMC_USE_BLAS 编译时，路由到外部 BLAS dgemm；否则使用内置分块三重循环（块大小 64）。

参数
[in]alpha标量乘子 α。
[in]A输入矩阵 A（不被修改）。
[in]transA非零表示对 A 取转置。
[in]B输入矩阵 B（不被修改）。
[in]transB非零表示对 B 取转置。
[in]beta标量乘子 β。beta==0 时 C 的旧值被忽略（可含 NaN）。
[in,out]C输出矩阵，维度须为 M×N。C 不可与 A 或 B 别名。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL。
  • LMMC_STATUS_DIMENSION_MISMATCH — op(A) 的列数 ≠ op(B) 的行数，或 C 维度不匹配。
副作用
  • 就地修改 C->data。A、B 不被修改。无内存分配。

详细描述

内存分配抽象层，可在标准库 malloc 与 LAMMP 栈分配器之间切换。

当定义 LMMC_USE_LAMMP_ALLOC 时，全部分配 / 释放路由到 LAMMP； 否则使用 malloc / free 。

在文件 memory_bridge.h 中定义.

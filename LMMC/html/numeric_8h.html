详细描述

数值常量、特殊值、近似比较与基本数学补充函数。

此模块包含 IEEE-754 特殊值生成 / 检测、扩展三角与对数函数、 浮点拆分、近似相等判定，以及一个轻量的 radix-4 FFT 与 Lambert W 函数。

同时考虑绝对与相对容差的近似相等判定。

[out]out_equal输出比较结果：1 表示近似相等，0 表示不等。

◆ lmmc_fft_radix4()

lmmc_status_t lmmc_fft_radix4 ( lmmc_real_t *  real,
lmmc_real_t *  imag,
size_t  n,
int  inverse 
)

严格 radix-4 就地 FFT。

对长度为 4 的幂的复数序列执行快速傅里叶变换。变换结果就地写回输入数组。 逆变换时自动除以 N 进行归一化。

参数
[in,out]real实部数组，长度 n ；变换后就地覆盖为频域实部。
[in,out]imag虚部数组，长度 n ；变换后就地覆盖为频域虚部。
[in]nFFT 长度，必须为 4 的幂。
[in]inverse非 0 时执行逆变换并归一化（除以 n）。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 n 不是 4 的幂或指针为 NULL。
副作用
  • 就地修改 real 和 imag 数组的全部 n 个元素。
  • 不分配堆内存。
  • real 和 imag 不可指向重叠的内存区域。

引用了 LMMC_STATUS_INVALID_ARGUMENT.

  • N 为 4 的幂 → radix-4 快速路径
  • N 为 2 的幂 → radix-2 Cooley-Tukey
  • 其他 → Bluestein chirp-z 算法（内部分配辅助缓冲区）
参数
[in,out]real实部数组，长度 n ；变换后就地覆盖。
[in,out]imag虚部数组，长度 n ；变换后就地覆盖。
[in]nFFT 长度，任意正整数。
[in]inverse非 0 时执行逆变换并归一化（除以 n）。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 n == 0 或指针为 NULL； ::LMMC_STATUS_ALLOC_FAILED 若 Bluestein 路径内存分配失败。
副作用
  • 就地修改 real 和 imag 数组的全部 n 个元素。
  • 当 n 非 2 的幂时，内部通过 Bluestein 算法分配临时缓冲区，函数返回前释放。
  • real 和 imag 不可指向重叠的内存区域。

使用递推关系将 x 提升到大值区域后应用渐近展开。

参数
[in]x输入值，必须为正数。
[out]out输出 \(\psi(x)\) 。
返回
LMMC_STATUS_OK 成功；LMMC_STATUS_INVALID_ARGUMENT 若 x ≤ 0 或 out 为 NULL。

计算双伽马函数 \(\psi(x) = \frac{d}{dx}\ln\Gamma(x)\) ，要求 \(x > 0\) 。

算法：

  1. 对 x < 6 使用递推关系 psi(x+1) = psi(x) + 1/x 将 x 提升到 >= 6。
  2. 对 x >= 6 使用渐近展开： psi(x) ~ ln(x) - 1/(2x) - sum_{k=1}^{N} B_{2k}/(2k * x^{2k}) 其中 B_{2k} 是 Bernoulli 数。

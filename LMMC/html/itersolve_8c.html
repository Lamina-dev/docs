◆ lmmc_itersolve_default_config()

lmmc_status_t lmmc_itersolve_default_config ( size_t  problem_size,
lmmc_itersolve_config_t *  out_cfg 
)

根据问题规模生成迭代求解器默认配置。

默认值：abs_tol=1e-12, rel_tol=1e-8, max_iter=max(100, 20*n), restart=min(n,30)。

参数
[in]problem_size问题维度 n，必须 > 0。
[out]out_cfg输出配置结构体。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — problem_size==0 或 out_cfg==NULL。
副作用
  • 覆写 out_cfg 的所有字段。无内存分配。

共轭梯度法（CG）求解对称正定稀疏系统 A*x = b。

要求 A 对称正定。x 作为初始猜测输入，收敛后存储解。 收敛判据：||r||_2 <= abs_tol + rel_tol * ||b||_2。

参数
[in]a对称正定稀疏矩阵（不被修改）。
[in]b右端向量（不被修改）。
[in]precond预处理子，可为 NULL（无预处理）。
[in]cfg配置参数，可为 NULL（使用默认值）。
[in,out]x初始猜测 / 输出解向量。
[out]out_result迭代统计结果，可为 NULL。
返回
  • LMMC_STATUS_OK — 成功（含收敛和达到最大步数两种情况，通过 out_result->converged 区分）。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL 或维度为 0。
  • LMMC_STATUS_DIMENSION_MISMATCH — 矩阵/向量/预处理子维度不匹配。
  • LMMC_STATUS_NUMERICAL_FAILURE — 出现 NaN/Inf 或分母退化。
副作用
  • 就地修改 x->data。分配并释放 4 个长度为 n 的临时向量。

重启型 GMRES 求解一般非对称稀疏系统 A*x = b。

使用 Arnoldi 过程构建 Krylov 子空间，Givens 旋转求解最小残差。 重启长度由 cfg->restart 控制（默认 30）。

参数
[in]a稀疏矩阵（不被修改）。
[in]b右端向量（不被修改）。
[in]precond预处理子，可为 NULL。
[in]cfg配置参数，可为 NULL。cfg->restart 控制重启长度。
[in,out]x初始猜测 / 输出解向量。
[out]out_result迭代统计结果，可为 NULL。
返回
同 ::lmmc_cg_solve。
副作用
  • 就地修改 x->data。
  • 分配 (restart+1) 个长度为 n 的 Arnoldi 基向量 + Hessenberg 矩阵等工作空间。

详细描述

迭代求解器使用的预处理子接口。

当前支持：恒等（无预处理）、Jacobi（对角）、ILU(0) 和 ILUT 。

◆ lmmc_precond_create_jacobi()

lmmc_status_t lmmc_precond_create_jacobi ( const lmmc_sparse_mat_t *  a,
lmmc_precond_t *  out_precond 
)

由稀疏矩阵的对角线创建 Jacobi 预处理子：M = diag(A)。

提取 A 的对角线元素并取倒数存储。若某对角元为零或过小（<1e-15）， 返回 SINGULAR_MATRIX。

参数
[in]a输入稀疏矩阵（CSR 格式，不被修改）。
[out]out_precond输出预处理子句柄。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL 或结构非法。
  • LMMC_STATUS_DIMENSION_MISMATCH — 矩阵非方阵。
  • LMMC_STATUS_SINGULAR_MATRIX — 对角线存在零元素。
  • LMMC_STATUS_ALLOCATION_FAILED — 内存分配失败。
副作用
  • 分配堆内存存储对角线倒数。调用方需配对调用 lmmc_precond_destroy 释放。

创建 ILU(0) 预处理子（零填充不完全 LU 分解）。

填充模式与输入矩阵 A 完全一致（不产生新的非零位置）。 内部复制 A 的结构并就地执行 IKJ 版本的 ILU 分解。

参数
[in]a输入稀疏矩阵（CSR 格式，不被修改）。
[out]out_precond输出预处理子句柄。
返回

  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL 或结构非法。
  • LMMC_STATUS_DIMENSION_MISMATCH — 矩阵非方阵。
  • LMMC_STATUS_SINGULAR_MATRIX — 对角线为零或缺失。
  • LMMC_STATUS_NUMERICAL_FAILURE — 分解过程中出现 NaN/Inf。
  • LMMC_STATUS_ALLOCATION_FAILED — 内存分配失败。
副作用

◆ lmmc_precond_apply()

lmmc_status_t lmmc_precond_apply ( const lmmc_precond_t *  precond,
const lmmc_vec_t *  rhs,
lmmc_vec_t *  out 
)

应用预处理子：求解 M*out = rhs。

根据 precond->type 分派到对应的求解逻辑：

  • NONE：直接复制 rhs 到 out。
  • JACOBI：逐元素乘以对角线倒数。
  • ILU0/ILUT：前代 + 回代。
参数
[in]precond预处理子句柄（不被修改）。
[in]rhs右端向量（不被修改）。
[out]out输出向量，长度须与 rhs 相同，内容被覆写。out 不可与 rhs 别名。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL。
  • LMMC_STATUS_DIMENSION_MISMATCH — 向量长度与预处理子尺寸不匹配。
副作用
  • 覆写 out->data。无内存分配。

详细描述

伪随机数发生器与常用分布。

内部使用线程不安全的 PCG/xoshiro 类生成器（不透明实现）， 调用方应为每个线程独立创建 lmmc_rng_t 。

◆ lmmc_rng_create()

lmmc_status_t lmmc_rng_create ( lmmc_rng_t **  out_rng)

创建随机数发生器实例。

分配并初始化一个 xoshiro256** 发生器，初始种子由系统时间或固定值确定 （具体取决于编译配置）。生成器为线程不安全，多线程环境下每个线程应 独立创建或使用 lmmc_rng_clone + lmmc_rng_jump 拆分子流。

参数
[out]out_rng输出新创建的 RNG 句柄。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 out_rng 为 NULL； ::LMMC_STATUS_ALLOC_FAILED 若内存分配失败。
副作用
  • 分配堆内存存储 RNG 内部状态，调用方必须调用 lmmc_rng_destroy 释放。

◆ lmmc_rng_gamma()

lmmc_status_t lmmc_rng_gamma ( lmmc_rng_t *  rng,
lmmc_real_t  shape,
lmmc_real_t  scale,
lmmc_real_t *  out 
)

生成 Gamma 分布样本 \(\mathrm{Gamma}(\alpha, \beta)\) 。

使用 Marsaglia-Tsang 方法（shape >= 1），shape < 1 时使用 \(X = Y \cdot U^{1/\alpha}\) 变换法（Y ~ Gamma(1+α, β)）。

参数
[in]rng已初始化的 RNG。
[in]shape形状参数 \(\alpha\) （> 0）。
[in]scale尺度参数 \(\beta\) （> 0）。
[out]out输出样本值。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 shape <= 0、scale <= 0 或指针为 NULL。
副作用
  • 就地修改 rng 的内部状态（消耗多个随机数，次数不确定）。
  • 不分配堆内存。

◆ lmmc_rng_poisson()

lmmc_status_t lmmc_rng_poisson ( lmmc_rng_t *  rng,
lmmc_real_t  lambda,
size_t *  out 
)

生成 Poisson 分布样本 \(\mathrm{Poisson}(\lambda)\) 。

lambda >= 10 使用 PTRD（Hörmann）算法（接受-拒绝）， lambda < 10 使用逆变换法（Knuth 方法）。

参数
[in]rng已初始化的 RNG。
[in]lambda期望值（> 0）。
[out]out输出样本值（非负整数）。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 lambda <= 0 或指针为 NULL。
副作用
  • 就地修改 rng 的内部状态（消耗随机数次数不确定，取决于接受-拒绝过程）。
  • 不分配堆内存。

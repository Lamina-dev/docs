特征值 / 奇异值分解、伪逆与条件数接口。
init.h
LMMC 库初始化与底层栈管理接口。
interp.h
一维插值算法：线性、三次样条、Lagrange 多项式、PCHIP、Akima、二维插值。
itersolve.h
稀疏线性系统的迭代求解器（CG / BiCGSTAB / GMRES / MINRES / LSQR）。
linear_algebra.h
稠密矩阵直接分解接口（LU、Cholesky、QR）。
nonlinear.h
标量非线性方程求根：二分法、Newton、割线法。
numeric.h
数值常量、特殊值、近似比较与基本数学补充函数。
ode.h
常微分方程初值问题求解器：Euler、RK4、RK45（自适应）。
precond.h
迭代求解器使用的预处理子接口。
quadrature.h
一元数值积分：梯形、Simpson、Gauss-Legendre、自适应 Simpson。
random.h
伪随机数发生器与常用分布。
sparse.h
稀疏矩阵数据结构、构造器与基本运算。
stats.h
基础统计量与组合数学接口。
status.h
LMMC 通用状态码定义。
tensor.h
N-D 稠密张量数据结构与基本运算。

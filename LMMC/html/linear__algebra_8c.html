◆ lmmc_lu_decompose_inplace()

lmmc_status_t lmmc_lu_decompose_inplace ( lmmc_mat_t *  a,
size_t *  pivots,
size_t *  out_swap_count 
)

就地 LU 分解（带部分主元选取）：P*A = L*U。

分解后 a 同时存储 L（严格下三角，单位对角线隐含）和 U（上三角含对角线）。 行交换记录在 pivots 数组中：pivots[k] 表示第 k 步与第 pivots[k] 行交换。

参数
[in,out]a输入方阵 A，输出为 LU 因子（就地覆写）。
[out]pivots置换数组，长度至少为 a->rows。
[out]out_swap_count主元交换次数（用于行列式符号），可为 NULL。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL 或矩阵非方阵。
  • LMMC_STATUS_SINGULAR_MATRIX — 主元绝对值 <= 1e-15，矩阵奇异。
副作用
  • 就地覆写 a->data（原始矩阵内容丢失）。
  • 覆写 pivots 数组。无堆内存分配。
使用模式
lmmc_lu_decompose_inplace(&A, pivots, NULL);
lmmc_lu_solve(&A, pivots, &b, &x);

◆ lmmc_lu_solve()

const size_t *  pivots,
const lmmc_vec_t *  b,

)

利用 LU 分解结果求解线性系统 A*x = b。

lu 和 pivots 须来自同一次 lmmc_lu_decompose_inplace 调用。 内部执行前代（L）+ 回代（U）+ 行置换应用。

参数
[in]luLU 因子矩阵（来自 decompose_inplace，不被修改）。
[in]pivots置换数组（来自 decompose_inplace）。
[in]b右端向量，长度须等于 lu->rows。
[out]x解向量，长度须等于 lu->rows，内容被覆写。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL。
  • LMMC_STATUS_DIMENSION_MISMATCH — 维度不匹配。
  • LMMC_STATUS_SINGULAR_MATRIX — U 对角线为零。
副作用
  • 覆写 x->data。无内存分配。lu 和 b 不被修改。

◆ lmmc_cholesky_decompose_inplace()

lmmc_status_t lmmc_cholesky_decompose_inplace ( lmmc_mat_t *  a)

就地 Cholesky 分解：A = L*L^T，要求 A 对称正定。

分解后 a 的下三角（含对角线）存储 L，上三角内容未定义。 使用相对阈值检测非正定：若某步对角元 <= rel_tol * max_diag 则判定失败。

参数
[in,out]a输入对称正定方阵，输出为下三角 L（就地覆写）。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL 或矩阵非方阵。
  • LMMC_STATUS_NOT_POSITIVE_DEFINITE — 矩阵非正定。
副作用
  • 就地覆写 a->data（原始矩阵内容丢失）。无堆内存分配。

副作用
  • 就地覆写 a->data 和 tau 数组。无堆内存分配。

◆ lmmc_qr_solve()

lmmc_status_t lmmc_qr_solve ( const lmmc_mat_t *  qr,
const lmmc_real_t *  tau,
const lmmc_vec_t *  b,
lmmc_vec_t *  x 
)

利用 QR 分解结果求解线性系统或最小二乘问题。

对 m×n 矩阵（m>=n）：求解 min||Ax-b||_2，即 R*x = Q^T*b 的前 n 行。 qr 和 tau 须来自同一次 lmmc_qr_decompose_inplace 调用。

参数
[in]qrQR 紧凑表示矩阵（不被修改）。
[in]tauHouseholder 标量数组。
[in]b右端向量，长度须等于 qr->rows。
[out]x解向量，长度须等于 qr->cols，内容被覆写。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL。
  • LMMC_STATUS_DIMENSION_MISMATCH — m < n 或向量长度不匹配。
  • LMMC_STATUS_SINGULAR_MATRIX — R 对角线为零。
  • LMMC_STATUS_ALLOCATION_FAILED — 临时工作内存分配失败。
副作用
  • 覆写 x->data。分配并释放长度为 m 的临时数组。

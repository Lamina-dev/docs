详细描述

True N-point FFT with radix-2 Cooley-Tukey and Bluestein chirp-z.

Provides lmmc_fft / lmmc_fft_forward / lmmc_fft_inverse for arbitrary N, plus the explicit padding helper lmmc_fft_radix4_pad_into.

Dispatch:

  • N is power of 4 -> existing radix-4 path (lmmc_fft_radix4)
  • N is power of 2 -> radix-2 Cooley-Tukey (in-place, iterative)
  • Otherwise -> Bluestein chirp-z (pad to next power-of-2 >= 2N-1)

lmmc_status_t lmmc_fft ( lmmc_real_t *  real,
lmmc_real_t *  imag,
size_t  n,

)

计算任意长度 N 点 DFT（正变换或逆变换）。

调度逻辑：

  • N 为 4 的幂 → radix-4 快速路径
  • N 为 2 的幂 → radix-2 Cooley-Tukey
  • 其他 → Bluestein chirp-z 算法（内部分配辅助缓冲区）
参数
[in,out]real实部数组，长度 n ；变换后就地覆盖。
[in,out]imag虚部数组，长度 n ；变换后就地覆盖。
[in]nFFT 长度，任意正整数。
[in]inverse非 0 时执行逆变换并归一化（除以 n）。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 n == 0 或指针为 NULL； ::LMMC_STATUS_ALLOC_FAILED 若 Bluestein 路径内存分配失败。
副作用
  • 就地修改 real 和 imag 数组的全部 n 个元素。
  • 当 n 非 2 的幂时，内部通过 Bluestein 算法分配临时缓冲区，函数返回前释放。
  • real 和 imag 不可指向重叠的内存区域。

◆ lmmc_sparse_create_csr()

lmmc_status_t lmmc_sparse_create_csr ( size_t  rows,
size_t  cols,
size_t  nnz,
lmmc_sparse_mat_t *  out_sparse 
)

创建指定 nnz 的空 CSR 矩阵（数组已分配但内容未初始化）。

调用方需手动填充 row_ptr、col_idx、values 数组后方可使用。 适用于已知稀疏结构的批量构造场景。

参数
[in]rows行数，必须 > 0。
[in]cols列数，必须 > 0。
[in]nnz非零元个数，可为 0。
[out]out_sparse输出矩阵，调用方需配对调用 lmmc_sparse_destroy 释放。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — rows/cols 为 0 或 out_sparse 为 NULL。
  • LMMC_STATUS_ALLOCATION_FAILED — 内存分配失败。
副作用
  • 分配堆内存（owns_data=1）。row_ptr 已清零，col_idx/values 内容未定义。

◆ lmmc_sparse_wrap_csr()

lmmc_status_t lmmc_sparse_wrap_csr ( size_t  rows,
size_t  cols,
size_t  nnz,
size_t *  row_ptr,
size_t *  col_idx,
lmmc_real_t *  values,
lmmc_sparse_mat_t *  out_sparse 
)

用外部缓冲区包装为 CSR 视图（不拥有内存，不分配）。

内部会验证 row_ptr 的单调性和 col_idx 的范围合法性。 owns_data 置 0，::lmmc_sparse_destroy 不会释放传入的缓冲区。

参数
[in]rows行数。
[in]cols列数。
[in]nnz非零元个数。
[in]row_ptr行指针数组，长度 rows+1，row_ptr[0]==0，row_ptr[rows]==nnz。
[in]col_idx列索引数组，长度 nnz，每个值 < cols。
[in]values数值数组，长度 nnz。
[out]out_sparse输出矩阵视图。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 参数非法或结构验证失败。
副作用
  • 无内存分配。对视图的写操作直接修改外部缓冲区。

lmmc_status_t lmmc_sparse_mat_mat_mul_dense ( const lmmc_sparse_mat_t *  sparse,
const lmmc_mat_t *  b,
lmmc_mat_t *  c 
)

稀疏矩阵 × 稠密矩阵：C = A * B（结果为稠密矩阵）。

◆ lmmc_nleq_newton()

lmmc_status_t lmmc_nleq_newton ( lmmc_opt_func_t  F,

void *  user_data,

const lmmc_optimize_config_t *  cfg,
lmmc_optimize_result_t *  out 
)

Newton 法求解非线性方程组 F(x) = 0。

每步求解 J(x)δ = -F(x)，通过 LU 分解。若 J 为 NULL， 使用前向有限差分近似 Jacobian。

参数
[in]F非线性函数回调。
[in]JJacobian 回调（可为 NULL，使用有限差分）。
[in]user_data传递给回调的用户数据。
[in,out]x初始猜测 / 输出解。
[in]cfg算法配置。
[out]out迭代结果。
返回
::LMMC_STATUS_OK（通过 out->converged 判断是否收敛）。
副作用
  • 就地修改 x->data。
  • 分配并释放临时向量（Fx, delta）和矩阵（Jmat, Jlu）+ pivots 数组。
  • 多次调用 F 和 J 回调。

Broyden 法求解非线性方程组 F(x) = 0。

维护 B ≈ J 的秩一更新：B += (ΔF - B·Δx)·Δxᵀ / (Δxᵀ·Δx)。 初始 B 通过有限差分近似。

参数
[in]F非线性函数回调。
[in]user_data传递给回调的用户数据。
[in,out]x初始猜测 / 输出解。
[in]cfg算法配置。
[out]out迭代结果。
返回
::LMMC_STATUS_OK（通过 out->converged 判断是否收敛）。
副作用
  • 就地修改 x->data。
  • 分配并释放多个临时向量和 n×n 矩阵。
  • 多次调用 F 回调。

Levenberg-Marquardt 非线性最小二乘。

求解 (JᵀJ + λI)δ = -Jᵀr，自适应调整 λ（接受步减小 λ，拒绝步增大 λ）。

参数
[in]residual残差函数回调 r(x)。
[in]JJacobian 回调（可为 NULL，使用有限差分）。
[in]user_data传递给回调的用户数据。
[in,out]x初始猜测 / 输出解。
[in]cfg算法配置（lm_damping 指定初始 λ）。
[out]out迭代结果。
返回
::LMMC_STATUS_OK（通过 out->converged 判断是否收敛）。
副作用
  • 就地修改 x->data。
  • 分配并释放多个临时向量和 n×n 矩阵。
  • 多次调用 residual 和 J 回调。

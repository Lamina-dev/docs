详细描述

稀疏矩阵数据结构、构造器与基本运算。

提供 CSR / CSC 两种压缩存储格式以及临时使用的 COO 三元组格式， 涵盖建造器（builder）、SpMV、SpGEMM、转置、格式互转、稀疏 LU / Cholesky 分解等接口。

struct lmmc_sparse_mat_t

通用稀疏矩阵（CSR / CSC 共用同一结构）。

当 format = LMMC_SPARSE_CSR ：

  • row_ptr 长度为 rows+1 ，
  • col_idx 与 values 长度均为 nnz 。

当 format = LMMC_SPARSE_CSC ：

  • row_ptr 在此别名为 col_ptr，长度为 cols+1 ，
  • col_idx 在此别名为 row_idx，长度为 nnz 。

创建稀疏矩阵增量构造器。

适用于非零元个数未知或乱序插入的场景。构造器内部维护动态数组， 容量不足时自动翻倍扩容。最终调用 lmmc_sparse_builder_build 生成 压缩格式矩阵。

参数
[in]rows矩阵行数，必须 > 0。
[in]cols矩阵列数，必须 > 0。
[in]initial_capacity初始预分配条目数；0 时使用内部默认值 16。
[out]out_builder返回的构造器句柄指针。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — rows/cols 为 0 或 out_builder 为 NULL。
  • LMMC_STATUS_ALLOCATION_FAILED — 内存分配失败。
副作用
  • 分配堆内存。调用方必须配对调用 lmmc_sparse_builder_destroy 释放。

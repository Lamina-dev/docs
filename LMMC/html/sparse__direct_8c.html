lmmc_status_t lmmc_sparse_lu_symbolic ( const lmmc_sparse_mat_t *  a,
lmmc_sparse_lu_t **  out_lu 
)

稀疏 LU 符号分析阶段：AMD 重排序 + 消去树构建。

分析矩阵的稀疏结构，计算 AMD（近似最小度）列置换以减少填充， 并预分配 L/U 因子的存储空间。分析结果可复用于多次数值分解 （当矩阵结构不变、仅数值变化时）。

参数
[in]a输入方阵（不被修改），rows 须等于 cols。
[out]out_lu返回的 LU 上下文句柄，调用方需配对调用 lmmc_sparse_lu_destroy 释放。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 指针为 NULL 或矩阵非方阵。
  • LMMC_STATUS_ALLOCATION_FAILED — 内存分配失败。
副作用
  • 分配堆内存（LU 上下文 + 内部工作数组）。a 不被修改。
  • 若 a 为 CSR 格式，内部会临时转换为 CSC 并在返回前释放。

lmmc_status_t lmmc_sparse_lu_numeric ( const lmmc_sparse_mat_t *  a,
lmmc_sparse_lu_t *  lu 
)

Gilbert-Peierls sparse LU factorization with partial pivoting.

稀疏 LU 数值分解阶段（Gilbert-Peierls 算法 + 部分主元）。

For each column k (in permuted order):

  1. Scatter permuted column of A into dense workspace.
  2. Use depth-first reach in the partially-built L to find the non-zero pattern of column k of L and U.
  3. Perform numeric update using the symbolic pattern.
  4. Apply partial pivoting.
  5. Store L and U entries.

lmmc_status_t lmmc_sparse_lu_solve ( const lmmc_sparse_lu_t *  lu,
const lmmc_vec_t *  b,
lmmc_vec_t *  x 
)

Solve Ax = b using the factored LU with AMD permutation.

利用已分解的稀疏 LU 因子求解 A*x = b。

The factorization is P*A*Q = L*U where: P = row permutation (partial pivoting) Q = column permutation (AMD)

Solve: A*x = b => P*A*Q * Q^{-1}*x = P*b L*U*y = P*b where y = Q^{-1}*x x = Q*y

N-D 稠密张量数据结构与基本运算。

支持任意阶（1 至 LMMC_TENSOR_MAX_NDIM）的张量， 以及向后兼容的三阶张量接口。

◆ lmmc_tensor_create()

lmmc_status_t lmmc_tensor_create ( size_t  ndim,
const size_t *  dims,
lmmc_tensor_nd_t *  out 
)

创建 N-D 张量，行优先连续存储，元素初始化为零。

分配 dims[0] × dims[1] × ... × dims[ndim-1] 个元素的连续缓冲区， 并设置行优先步距。输出张量的 owns_data 为 1。

参数
[in]ndim维度数，范围 [1, LMMC_TENSOR_MAX_NDIM]。
[in]dims各维大小数组，长度 ndim ，每个元素须 >= 1。
[out]out输出张量结构体（调用方提供存储）。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 ndim 超出范围、dims 含零或指针为 NULL； ::LMMC_STATUS_ALLOC_FAILED 若内存分配失败。
副作用
  • 分配堆内存用于数据缓冲区（out->data），调用方必须调用 lmmc_tensor_nd_destroy 释放。
  • out 结构体的所有字段被覆盖写入。

◆ lmmc_tensor_permute()

lmmc_status_t lmmc_tensor_permute ( const lmmc_tensor_nd_t *  in,
const size_t *  perm,
lmmc_tensor_nd_t *  out 
)

对张量维度进行置换（转置的推广），产生新的拥有数据的张量。

按照 perm 指定的轴顺序重新排列数据。例如 perm={1,0,2} 交换前两个轴。 输出张量拥有独立的数据副本（owns_data == 1）。

参数
[in]in输入张量。
[in]perm置换数组，长度 in->ndim，为 [0, ndim) 的排列。
[out]out输出张量结构体（调用方提供存储）。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若 perm 不是有效排列或指针为 NULL； ::LMMC_STATUS_ALLOC_FAILED 若内存分配失败。
副作用
  • 分配堆内存用于输出数据缓冲区，调用方必须调用 lmmc_tensor_nd_destroy 释放。
  • 不修改输入张量 in 。

◆ lmmc_tensor_nd_reshape_view()

lmmc_status_t lmmc_tensor_nd_reshape_view ( const lmmc_tensor_nd_t *  src,
size_t  new_ndim,
const size_t *  new_dims,
lmmc_tensor_nd_t *  out_view 
)

N-D 张量重塑视图（零拷贝），返回共享底层缓冲区的非拥有视图。

仅当源张量为连续存储（行优先步距）时可成功。新形状的元素总数必须与源相同。 输出视图的 owns_data == 0，生命周期不得超过源张量。

参数
[in]src源张量（必须为连续存储）。
[in]new_ndim新维度数，范围 [1, LMMC_TENSOR_MAX_NDIM]。
[in]new_dims新各维大小数组，长度 new_ndim 。
[out]out_view输出视图结构体。
返回
LMMC_STATUS_OK 成功； LMMC_STATUS_INVALID_ARGUMENT 若元素总数不匹配、源非连续或指针为 NULL。
副作用
  • 不分配堆内存。输出视图共享源张量的 data 指针。
  • 修改源张量数据会同时影响视图，反之亦然（别名关系）。
  • 源张量销毁后视图将悬空，调用方须保证生命周期正确。

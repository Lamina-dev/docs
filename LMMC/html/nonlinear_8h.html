◆ lmmc_bisection_solve()

lmmc_status_t lmmc_bisection_solve ( lmmc_scalar_func_t  func,
void *  user_data,
lmmc_real_t  left,
lmmc_real_t  right,
const lmmc_nonlinear_config_t *  cfg,
lmmc_nonlinear_result_t *  out_result 
)

二分法求根。

要求 f(left)*f(right) < 0（区间端点异号）。每步将区间减半， 收敛判据：|f(mid)| <= abs_tol 或半区间宽度 <= x_tol。

参数
[in]func标量函数 f(x)。
[in]user_data传递给 func 的用户上下文。
[in]left区间左端点，要求 left < right。
[in]right区间右端点。
[in]cfg配置参数，可为 NULL（使用默认值）。
[out]out_result求根结果统计。
返回
  • LMMC_STATUS_OK — 成功（含收敛情况）。
  • LMMC_STATUS_INVALID_ARGUMENT — 参数非法或区间不变号。
  • LMMC_STATUS_NUMERICAL_FAILURE — 出现 NaN/Inf 或达到最大步数。
副作用
  • 无内存分配。多次调用 func。

◆ lmmc_newton_solve()

lmmc_status_t lmmc_newton_solve ( lmmc_scalar_func_t  func,
lmmc_scalar_dfunc_t  dfunc,
void *  user_data,
lmmc_real_t  x0,
const lmmc_nonlinear_config_t *  cfg,
lmmc_nonlinear_result_t *  out_result 
)

Newton 法求根。

若 dfunc 为 NULL，使用中心差分自动计算导数。 收敛判据：|f(x)| <= abs_tol 或 |x_{k+1}-x_k| <= x_tol。 导数过小时报告 ZERO_DERIVATIVE 失败。

参数
[in]func标量函数 f(x)。
[in]dfunc解析导数 f'(x)，可为 NULL（自动差分）。
[in]user_data传递给 func/dfunc 的用户上下文。
[in]x0初始猜测。
[in]cfg配置参数，可为 NULL。
[out]out_result求根结果统计。
返回
  • LMMC_STATUS_OK — 成功。
  • LMMC_STATUS_INVALID_ARGUMENT — 参数非法。
  • LMMC_STATUS_NUMERICAL_FAILURE — 数值问题或未收敛。
副作用
  • 无内存分配。多次调用 func（和 dfunc）。

[out]out_result求根结果统计。

副作用

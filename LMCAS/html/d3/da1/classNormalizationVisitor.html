std::shared_ptr< SymbolicNode > NormalizationVisitor::try_assumption_simplify ( const std::shared_ptr< SymbolicNode > &  node)
inlineprivate

Attempt assumption-based simplification on a node.

Applies the following rules when an AssumptionContext is available:

  • sqrt(x²) → x when x is NonNegative
  • sqrt(x²) → abs(x) when x is Real (but not NonNegative)
  • abs(x) → x when x is Positive
  • abs(x) → -x when x is Negative
参数
nodeThe node to attempt simplification on
返回
Simplified node if a rule applied, or nullptr if no rule matched

详细描述

RelationStore class for storing relational constraints between symbolic expressions.

Stores inequality relations (GT, LT, GEQ, LEQ, NEQ) between SymbolicExpr instances. When a simple "variable op 0" pattern is detected, the corresponding sign property is automatically propagated to the PropertyStore.

174/**
175 * @brief 三角组合积分策略：处理 sin^m(x)·cos^n(x)、tan^n(x)、sec^n(x) 形式。
176 *
177 * 当被积函数形如：
178 * - sin^m(var)·cos^n(var)（m,n 为非负整数，m+n ≤ 8），或
179 * - tan^n(var)（n 为整数，2 ≤ n ≤ 8），或
180 * - sec^n(var)（n 为偶数，2 ≤ n ≤ 8）
181 * 时，使用 Pythagorean 恒等式与降幂公式得到闭式原函数：
182 * - 奇数次幂情形（m 或 n 为奇数）通过 u = cos(x) 或 u = sin(x) 换元求解；
183 * - 双偶数情形（m,n 均为偶数）通过半角恒等式 sin² = (1-cos2x)/2 / cos² = (1+cos2x)/2 递归降阶；
184 * - tan^n 通过 tan^n = tan^(n-2)·(sec²-1) 递归化简；
185 * - 偶数 sec^n 通过分部积分得到的递推公式化简。
186 *
187 * sin/cos/tan/sec 的内部参数必须恰为积分变量本身（如 sin(2x) 由 LinearSubstitutionStrategy 处理）。
188 * 若指数为非整数、负数或超出 m+n > 8 的总度数限制，则返回 nullptr 让链上后续策略尝试。
189 */

201 * @param n_out cos 的幂次（非负整数）
202 * @return 匹配成功返回 true；若包含其他因子、参数不为 var、或幂次为非整数/负数则返回 false

230 * @brief 根据 m,n 奇偶性分派到 odd / even 情形的内部分发器（带尺度参数）。
231 * @param m sin 幂次

248 /**

251 * 将被积函数展开为 cos^k(2c·x) 的多项式，并对每个 k 递归调用

274/**
275 * @brief 高次有理函数部分分式积分策略：处理分母次数 ≥ 3 的有理函数 P(x)/Q(x)。
276 *
277 * 工作流程：
278 * 1. extract_rational：将被积表达式分解为 P(x)/Q(x)，要求 P、Q 均为关于积分变量
279 * 的多项式，系数为有理数；非有理形式时返回 false。
280 * 2. poly_divide：当 deg(P) ≥ deg(Q) 时进行多项式长除法，得到商和余式；商部分用幂律
281 * 逐项积分，剩余的真分式进入部分分式分解。
282 * 3. factor_denominator：用 square_free_factorization 加上有理根定理，把 Q 分解为
283 * 线性因子和不可约二次因子（在有理数域上）。无法在有理数上完成分解时返回 false。
284 * 4. solve_coefficients：对每个因子按重数引入待定系数（线性因子贡献 A_l/(x-r)^l，
285 * 二次因子贡献 (B_l·x + C_l)/q(x)^l），通过比较多项式系数列出线性方程组，使用
286 * gaussian_eliminate 求解。
287 * 5. integrate_term：逐项积分：
288 * - A/(x-r)：A·ln|x-r|；A/(x-r)^l (l≥2)：-A/((l-1)(x-r)^(l-1))；
289 * - (Bx+C)/q(x)：分裂为 ln 和 arctan 两部分（完全平方）；
290 * - 不可约二次因子的高次幂 q^l (l≥2) 暂以未求值积分节点保留。
291 *
292 * 失败保护：
293 * - 分母 Q 为零多项式时直接返回未求值积分节点；
294 * - 表达式不是有理函数时返回 nullptr，让链上后续策略尝试；
295 * - 因式分解或线性方程组无解时返回未求值积分节点保留原积分形式。
296 */
297class LAMINA_API RationalDecompositionStrategy : public IntegrationStrategy {
298public:
299 std::shared_ptr<SymbolicExpr> try_integrate(
300 const SymbolicExpr& expr, const std::string& var, Integrator& ctx, int depth = 0) override;
301 std::string name() const override { return "RationalDecomposition"; }
302
303 /**
304 * @brief 把被积表达式拆解为 P(x)/Q(x)（P、Q 均为有理系数多项式）。
305 * @param expr 被积表达式
306 * @param var 积分变量名
307 * @param P_out 输出参数：分子多项式
308 * @param Q_out 输出参数：分母多项式
309 * @return 成功识别为有理函数返回 true；表达式含 sin/exp/log/非整数指数等
310 * 非多项式部分时返回 false
311 */
312 bool extract_rational(const SymbolicExpr& expr, const std::string& var,
313 Polynomial<Rational>& P_out,
314 Polynomial<Rational>& Q_out);
315
316 /**
317 * @brief 多项式长除法：P = Q·quotient + remainder。
318 * @param P 被除多项式
319 * @param Q 除多项式（不能为零）
320 * @param quotient_out 输出：商多项式
321 * @param remainder_out 输出：余式多项式（deg(remainder) < deg(Q)）
322 */
323 void poly_divide(const Polynomial<Rational>& P, const Polynomial<Rational>& Q,
324 Polynomial<Rational>& quotient_out,
325 Polynomial<Rational>& remainder_out);
326
327 /**
328 * @brief 把 Q 分解为线性 (x-r) 与不可约二次 (x²+px+q) 因子的集合。
329 * @param Q 分母多项式
330 * @param factors_out 输出：(因子, 重数) 列表；每个因子是首一线性或首一不可约二次
331 * @return 全部分解为线性/不可约二次因子返回 true；含有更高次不可约因子或
332 * 有理根定理无法穷尽线性因子时返回 false
333 */
334 bool factor_denominator(const Polynomial<Rational>& Q,
335 std::vector<std::pair<Polynomial<Rational>, int>>& factors_out);
336
337 /**
338 * @brief 求解部分分式各项的待定系数。
339 *
340 * 对线性因子 (x-r) 重数 m，引入 m 个常数系数 A_1..A_m；对不可约二次因子 q(x)
341 * 重数 m，引入 m 对线性系数 (B_l, C_l)。通过比较多项式系数构造方阵并用
342 * gaussian_eliminate 求解。
343 *
344 * @param P 部分分式分解的目标多项式（真分式的分子，deg(P) < deg(Q)）
345 * @param Q 部分分式分解的目标分母
346 * @param factors 因子分解（来自 factor_denominator）
347 * @param numerators_out 输出：与 (factor_index, l) 一一对应的分子多项式列表，
348 * 每个分子要么是常数 (A)，要么是一次多项式 (B·x + C)
349 * @return 系数解唯一存在时返回 true；系统无解或退化时返回 false
350 */
351 bool solve_coefficients(const Polynomial<Rational>& P,
352 const Polynomial<Rational>& Q,
353 const std::vector<std::pair<Polynomial<Rational>, int>>& factors,
354 std::vector<Polynomial<Rational>>& numerators_out);
355
356 /**
357 * @brief 对单个部分分式项 numerator(x)/factor(x)^power 求积分。
358 *
359 * 支持的形式：
360 * - factor 是 x-r：power=1 → A·ln(x-r)；power≥2 → -A/((power-1)·(x-r)^(power-1))
361 * - factor 是不可约二次 x²+px+q：power=1 → 拆为 (B/2)·ln(q(x)) +
362 * (D-Bp/2)·(2/√(4q-p²))·arctan((2x+p)/√(4q-p²))；power≥2 暂以未求值积分节点返回
363 *
364 * @param numerator 分子多项式（线性因子时为常数；二次因子时为 B·x+C）
365 * @param factor 单个因子多项式（首一线性或首一不可约二次）
366 * @param power 因子的幂指数（≥1）
367 * @param var 积分变量名
368 * @return 该项的原函数表达式
369 */
370 std::shared_ptr<SymbolicExpr> integrate_term(
371 const Polynomial<Rational>& numerator,
372 const Polynomial<Rational>& factor,
373 int power, const std::string& var);
374};

504 *
505 * 给定一个被积表达式和一组按内层（index 0）到外层（最后一个 index）排序的
506 * IntegrationStep（每一步包含变量名、可选的下界与上界），引擎会依次对每个
507 * 变量调用 Integrator::integrate（不定积分，下/上界均为 nullptr）或
508 * Integrator::integrate_def（定积分，下/上界均给出）。如果某一步产生未求值
509 * 的积分节点（Calculus_Integral），引擎会停止后续步骤，返回当前的部分结果。
510 *
511 * 当被积表达式不依赖于某个具有定积分边界 [a,b] 的变量时，引擎会直接将当前
512 * 表达式乘以 (b - a)，避免不必要的形式积分。
513 *
514 * 输入约束：步骤数为 1~3；变量名不能重复；每一步的 lower/upper 必须同时为
515 * nullptr（不定积分）或同时非空（定积分）。任何违反约束的输入返回 nullptr。
516 */

518public:

520 struct IntegrationStep {
521 std::string variable; ///< 积分变量名
522 std::shared_ptr<SymbolicExpr> lower; ///< 下界，nullptr 表示不定积分
523 std::shared_ptr<SymbolicExpr> upper; ///< 上界，nullptr 表示不定积分
524 };

530 * @param integrator 用于实际执行单变量积分的 Integrator 实例
531 * @return 迭代积分结果；若输入非法或任一步骤未能求值则可能返回 nullptr 或
532 * 保留未求值积分节点的部分结果
533 */
